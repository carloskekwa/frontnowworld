
A cart inventory CRUD web app in this app a user can do the following things.
- Add a cart
- Read a cart
- Edit a cart
- Delete a cart



## How to run the app?
- Run `npm install` to install required dependencies.
- Run `ng serve` to run the angular app

## Built with
- Node 10.15.0
- Angular 7.2.3
- Firebase 6.0.2
- Angular Material 7.3.7
